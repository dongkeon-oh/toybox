TB_ITEM -> ITM-ID 생성 규칙
-----------------------------------------------------------------------------------------------------
총 8자리로 ID를 생성

2자리 : 종류	
	ex) BG : 보드게임, ED : 전자장비 등등
4자리 : 고유번호
1자리 : 부속품 유무
	ex) C : 부속품 있음 (Bang의 경우 확장팩 등등), N : 부속품 없음
1자리 : 소유구분
	ex) I : 보드게임팀 소유, O : 개인 소유, E : 기타 경우
